Prueba Técnica - Desarrollador Jr. (UFPS)

Este repositorio contiene la solución a dos retos técnicos de lógica y algoritmos desarrollados en Python.
📁 Contenido del Proyecto
1. Agrupador de Anagramas (/01-anagramas)

    Reto: Agrupar palabras que comparten las mismas letras.

    Solución: Uso de diccionarios (Map) y ordenamiento de caracteres para una búsqueda eficiente.

    

2. Cola con Prioridad (/02-cola-con-prioridad)

    Reto: Simular la atención bancaria atendiendo primero a clientes VIP, luego Preferenciales y finalmente Regulares.

    Solución: Implementación manual de la clase CustomerQueue sin librerías externas.

    Lógica: Se respeta el orden de llegada para clientes con la misma prioridad.
